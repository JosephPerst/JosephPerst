City Island native. West Point cadet (Class of 2026) studying Applied Statistics & Data Science. I build ML systems for defense and industry.

## Selected Work

| Project | Description |
|---------|-------------|
| [Insider Threat Detection](https://github.com/JosephPerst/nlp-text-analysis) | NLP-based triage and risk-scoring for insider threat reports at the U.S. Department of Defense. Transformer models (BERT, BigBird, Longformer) designed for integration into Army Vantage. |
| [Revenue Forecasting](https://github.com/JosephPerst/parking-forecasting) | Predictive ML models for the Florida Panthers across F&B, merchandise, parking, and membership retention verticals. |
| [IR Image Captioning & VQA](https://github.com/JosephPerst/ir-image-captioning-vqa) | End-to-end pipeline for captioning and question answering on infrared (FLIR) imagery. Bridging civilian RGB VQA to military thermal imaging. |
| [Network Protocols](https://github.com/JosephPerst/networking-projects) | Router, switch, and TCP/UDP stack from scratch in Python. OSPF-style link-state routing with Dijkstra's algorithm. |
